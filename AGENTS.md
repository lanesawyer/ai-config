# Agent Instructions

## Style

- Concise responses — no trailing summaries, no narrating internal steps
- No comments in code unless the WHY is non-obvious
- No unsolicited refactors or abstractions beyond the task

## Commits

Follow conventional commits: `type(scope): message`. Types: feat, fix, refactor, chore, docs, test.

## Workflow

Skills (slash commands) are available in Claude Code for common workflows: start-work, ship, address-pr-comments, write-tests, code-review, and more. Prefer these over ad-hoc sequences when they fit.
